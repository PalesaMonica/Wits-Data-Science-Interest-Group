## Pandas & SQL (PostgreSQL) Basics
### Overview

This Section is designed to introduce Pandas and SQL (PostgreSQL) for data retrieval, exploration, and basic analysis.

By the end, you will be able to:

- Load, view, and inspect datasets with Pandas
- Select, filter, and summarize data
- Connect SQL queries to data analysis
- Understand the relationship between SQL and Pandas

## What You Will Learn
### Pandas Basics
- Loading datasets (.csv, .xlsx)
- Inspecting data (head(), tail(), shape, info(), describe())
- Selecting columns and rows
- Filtering and conditional selection
- Aggregations (mean, sum, groupby)
- Basic data exploration and summary statistics

### SQL Basics (PostgreSQL)
- Connecting to a PostgreSQL database
- Writing simple queries: SELECT, WHERE, GROUP BY, ORDER BY
- Filtering, aggregating, and summarizing data

Exporting query results to CSV for use in Pandas

### Pandas ↔ SQL Relationship
- SQL queries = asking questions from databases
