## Wits Data Science Interest Group
### Overview

The Wits Data Science Interest Group is a student-led community focused on building practical data skills that bridge the gap between academic learning and industry careers.

Our goal is to help members develop real-world data competencies through hands-on practice, industry-aligned tools, and career-focused learning.

This group is suitable for students interested in pursuing careers as:

- Data Analysts
- Data Scientists
- Machine Learning Engineers
- Data Engineers

## Our Mission

- Build solid foundations in data handling, analysis, and engineering
- Connect theoretical concepts to real-world industry use cases
- Encourage practical learning through projects and collaboration
- Prepare members for internships, graduate roles, and research careers

## What We Focus On
### Core Data Skills 

- Python for data work
- Pandas for data manipulation
- SQL for querying databases
- Understanding data types and schemas
- Basic statistics and data exploration
- Data workflows used in industry

### Data Science & Machine Learning 

- Exploratory Data Analysis (EDA)
- Statistical thinking and hypothesis testing
- Feature engineering and data preprocessing
- Introduction to machine learning concepts: regression, classification, clustering
- Model evaluation and interpretation
- Data storytelling and insights communication

### Data Engineering 

- Databases and data modeling
- SQL pipelines
- Data extraction, transformation, and loading (ETL)
- Data workflow design in production systems
- Introduction to big data tools

## How We Learn

Our sessions are short, focused, and practical:

- Weekly 45-minute sessions
- Minimal theory, maximum practical exposure
- Realistic datasets and industry-style problems
- Collaborative learning and discussions

#### Each session answers:

*How is this skill used in real data jobs?*

## Learning Philosophy

We believe that:

- Strong data foundations matter more than advanced algorithms
- SQL and data handling are the core of most data roles
- Communication and interpretation are as important as coding skills
- Practical exposure supports all advanced specializations: Data Science, ML, and Data Engineering

## Tools & Technologies

Members will be exposed to tools commonly used in industry, including:
- Python
- Pandas & NumPy
- PostgreSQL
- Jupyter Notebooks
- Git & GitHub
- Data visualization libraries
- Introduction to machine learning libraries (Scikit-learn, TensorFlow, or PyTorch)
- PowerBI

## Outcomes for Members

By participating in this group, members will:
- Gain confidence working with real datasets
- Understand how data is stored, queried, and analyzed
- Build a foundation for multiple data-related careers
- Develop portfolio-ready projects
- Learn the basics of machine learning workflows
- Improve problem-solving and analytical thinking

## Who Can Join?

- Students with interest in data, machine learning, or analytics careers
- Beginners welcome ,no prior experience required
- Curiosity, consistency, and willingness to learn matter most
